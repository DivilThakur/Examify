# 🧑‍🏫 E-Learning Exam Platform

An online exam management system where **examiners** can create and manage exams, and **students** can register, take exams, and receive certificates upon successful completion. The platform supports role-based authentication and offers a streamlined dashboard for both examiners and students.

## 🔗 Live Demo

👉 **[Try it here](https://examify-eta.vercel.app/)** 

## 🚀 Features

- ✅ User registration and login for examiners and students
- 📝 Examiner dashboard to create exams and add questions
- 🎓 Student dashboard to view assigned exams and take them
- 📜 Automatic certificate generation for passed exams
- 🔒 Role-based access control and secure authentication

## 🛠️ Tech Stack

- Frontend: React 
- Backend: Node.js 
- Database: MongoDB 
- Authentication: JWT 

## 📦 Installation & Running Locally

1. Clone the repo:

```bash
git clone https://github.com/your-username/e-learning-platform.git
cd e-learning-platform

## Setup backend:
cd backend
npm install
npm start

## Setup frontend:
cd frontend
npm install
npm start
```

## 🎯 Usage
- Examiner registers and logs in

- Examiner creates new exams and adds questions

- Students register and log in

- Students see available exams in their dashboard and take exams

- Upon passing, students receive certificates
